# BootCamp Java

Ejercicios y ejemplos del bootcamp de Java. Patrones de diseño, conceptos orientados a objetos y prácticas fundamentales.

## Contenido

### Builder Pattern
Ejemplo práctico del patrón Builder para construir objetos `Building` paso a paso.

**Archivos:**
- `src/Builderexample/Building.java` — Entidad inmutable con constructor completo
- `src/Builderexample/BuildingBuilder.java` — Builder que permite configurar atributos incrementalmente
- `src/Builderexample/Main.java` — Demo interactiva por consola

## Cómo ejecutar

```bash
cd src/Builderexample
javac Main.java Building.java BuildingBuilder.java
java Main
```
