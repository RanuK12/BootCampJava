# HANDOFF — BootCamp Java

## Propósito
Repositorio de ejercicios y ejemplos del bootcamp de Java. Patrones de diseño, OOP y fundamentos del lenguaje.

## Estado actual
- ✅ Ejemplo del patrón Builder completo e interactivo
- 🚧 Requiere más ejercicios y patrones

## Stack
- Java 17+
- Patrón Builder implementado manualmente

## Qué funciona
- `BuildingBuilder` construye objetos `Building` paso a paso
- `Main` interactivo pide datos por consola

## Qué está roto / pendiente
- Solo hay 1 patrón (Builder)
- Falta documentación de otros conceptos de OOP
- No hay tests unitarios

## Próximos pasos
1. Agregar más patrones: Singleton, Factory, Strategy, Observer
2. Incluir tests con JUnit 5
3. Agregar ejercicios de colecciones y streams

## Notas para retomar
- Compilar con `javac` y ejecutar con `java` desde `src/Builderexample/`
- El proyecto usa estructura de paquetes por patrón
